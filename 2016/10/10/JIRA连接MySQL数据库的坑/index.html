JIRA连接MySQL数据库的坑

发表于 2016-10-10   |   分类于 技术   |  

JIRA是一个项目管理工具，跟进项目的进度，支持多种强悍的插件，比如，git,gitlab等，它是一种本地安装软件，但是提供web页面进行管理。

所有关于项目的数据都是保存到数据库的，JIRA支持多种关系型数据库例如MySQL、PostgreSQL、SQL Server，所以就涉及到连接数据库的问题，本文只谈怎么连MySQL。由于MySQL的版本更新的问题，导致连接过程中提示错误：Unknown system variable ‘storage_engine’。这是由于在MySQL 5.7.5 版本中去除了这个变量，用default_storage_engine代替，请见===》MySQL Server System Variables。
